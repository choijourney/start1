  	모든 단축키는 Window >> Preferences >> General >> Keys 메뉴에서 확인 및 변경이 가능하다.
변수, 메소드 이름 한번에 변경 : alt + Shift + R
alt + Shift + M : 블럭잡은 구문을 메소드로 추출 / 생성자생성 @@@@@@
Alt + UP,DOWN : 현재 줄 위치 이동
ctrl + space누르고 메서드이름 입력하면 오버라이딩 자동완성
Ctrl + Shift + G : 현재 커서가 위치한 특정 메써드나 객체, 필드를 Reference하고 있는 곳을 찾는다.
Ctrl + Shift + M : 캐럿이 위치한 대상에 필요한 특정 클래스 Import 시키기@@
Ctrl + Shift + O : 소스에 필요한 패키지를 자동으로 Import 시키기 / 안쓰는 import제거는 확인못함
Ctrl + shift + R : Open Resource. 모든 프로젝트에서 파일명 검색.
Ctrl + Shift + S : 열려진 모든파일 저장 및 컴파일 @@@
Ctrl + F11 : 실행  @@@@@@@@@
Ctrl + 2 + R : Rename (리팩토링)
Ctrl + 1 : Quick Fix. 에러가 발생했을 경우 Quick Fix를 통해 쉽게 해결이 가능하다.(Rename에 주로 사용)  import기능있다는데 확인못함 
Ctrl + Alt + up/down : 한줄 duplicate복사  @@@@@@@@
Ctrl + Shift + / : 선택 영역 Block Comment 블록주석 설정
Ctrl + Shift + \ : 선택 영역 Block Comment 블록주석 제거
Ctrl + . : 다음오류부분으로 가기
Ctrl + , : 이전오류부분으로 가기
Ctrl + D : 한줄삭제 @@
Ctrl + I : 소스 깔끔 정리(인덴트 중심의 자동구문정리) 인덴트=들여쓰기
Ctrl + H : Find 및 Replace 파일서치만 할수있는 방법도 있음
Ctrl + 객체클릭(혹은 F3) : 클래스나 메소드 혹은 멤버를 정의한 곳으로 이동(Open Declaration)
Alt + Shift + UP : 커서를 기준으로 토큰단위 블럭지정 (괄호의 열고 닫기 쌍 확인에 유용)  ==> 
괄호의 뒤에 마우스 커서를 위치시킨 후 더블클릭한 것과 같은 효과
Alt + Enter 해당 프로젝트에서. : Project 속성
F4 : 클래스명을 선택하고 누르면 해당 클래스의 Hierarchy 부모를 볼 수 있다.
Alt + <- : 이전 작업 화면
Alt + -> : 다음 작업 화면
Ctrl + Shift + T : 클래스 찾기
Ctrl + O : 모든 클래스, 프로퍼티 트리 보기, 선택가능
F12 : 컴파일 중 에러등으로 포커스가 다른데로 갔을 때 Editor 로 커서 이동
에디터 창으로 이동 : F12
Ctrl + 1 : 컴파일 에러가 발생한 곳에서 Ctrl + 1을 누를 경우 컴파일 에러에 대한 해결책을 제시
Ctrl + Shift + L : 단축키 보기 Hint

getter setter 창에서 alt + a 누르면 전체선택
대문자 변환 : Ctrl + Shift + X
소문자 변환 : Ctrl + Shift + Y

글자 크기 크게 : Ctrl + +
글자 크기 작게 : Ctrl + -
